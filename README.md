## Install the dependencies

```bash
poetry install
```
or 
```bash
pip install -e .
```

## Run the tests

```bash
pytest
```

## Run the tests with coverage

```bash
coverage run -m pytest
```

See the coverage report by:

```bash
coverage report
```

View this in HTML format:

```bash
coverage html
```

Then open the `htmlcov/index.html` file in your browser.
